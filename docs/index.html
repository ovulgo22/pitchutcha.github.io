Funcionalidades do Projeto

O projeto Pitchutcha oferece uma série de funcionalidades projetadas para facilitar o gerenciamento e a organização de informações. Aqui estão algumas delas:

  • Interface Amigável: A UI foi projetada com o usuário em mente, tornando a navegação intuitiva.
  • Configurações Personalizáveis: Os usuários podem adaptar a plataforma às suas necessidades, incluindo temas e notificações.
  • Exportação de Dados: Permite que os dados sejam exportados em formatos comuns como CSV, JSON e PDF.
  • Multiusuário: Suporte para múltiplos usuários, permitindo que diferentes pessoas acessem a plataforma com permissões variadas.

Abaixo estão algumas metas e desenvolvimentos planejados para o futuro do projeto Pitchutcha:

Testes e Validações

A qualidade do projeto é uma prioridade. Abaixo, descreveremos os tipos de testes realizados:

Testes Unitários

Testes unitários são implementados para validar cada componente isoladamente. Utilizamos Jest para a execução desses testes.

Testes de Integração

Os testes de integração garantem que diferentes módulos funcionem juntos corretamente. Ferramentas como Mocha e Chai são usadas.

Testes de Aceitação

Realizamos testes de aceitação com usuários finais para garantir que o produto atenda às necessidades do usuário.

Acessibilidade

Estamos comprometidos em garantir que nosso projeto seja acessível a todos os usuários. Aqui estão algumas práticas implementadas:

  • Texto Alternativo: Todas as imagens têm textos alternativos descritivos para ajudar usuários com deficiência visual.
  • Navegação por Teclado: Todas as seções são acessíveis via teclado, permitindo navegação sem a necessidade de um mouse.
  • Contraste Adequado: As cores do site foram escolhidas para garantir que o texto seja legível para todos, incluindo pessoas com daltonismo.
  • ARIALabels: Usamos ARIA roles e labels para melhorar a experiência de leitores de tela.

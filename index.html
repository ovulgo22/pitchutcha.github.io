Primeiros Passos

Bem-vindo à nossa API! Este guia irá ajudá-lo a começar rapidamente com nossa plataforma.

cURL 
curl -X GET "https://api.exemplo.com/v1/users" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"

Instalação

Para começar, você precisará de uma chave de API. Você pode obter uma em seu painel de controle.

ℹ️
Dica: Mantenha sua chave de API segura e nunca a compartilhe publicamente.

Configuração

Configure sua chave de API como uma variável de ambiente:

Bash 
export API_KEY="sua_chave_aqui"

Autenticação

Nossa API usa autenticação baseada em tokens Bearer. Inclua seu token no cabeçalho Authorization de todas as requisições.

JavaScript 
const response = await fetch('https://api.exemplo.com/v1/users', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
console.log(data);
⚠️
Atenção: Sempre use HTTPS em produção para proteger suas credenciais.

Usuários

Gerencie usuários em sua aplicação com nossos endpoints de usuários.

GET /v1/users

Lista todos os usuários com paginação.

Parâmetros de Query

Parâmetro Tipo Obrigatório Descrição
page integer Não Número da página (padrão: 1)
limit integer Não Itens por página (padrão: 20, máx: 100)

Resposta

JSON 
{
  "data": [
    {
      "id": "user_123",
      "name": "João Silva",
      "email": "joao@exemplo.com",
      "created_at": "2024-01-15T10:30:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 150,
    "pages": 8
  }
}

Exemplos

Exemplos práticos de como usar nossa API em diferentes linguagens.

Python 
import requests

# Configurar a chave da API
headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

# Fazer uma requisição
response = requests.get(
    'https://api.exemplo.com/v1/users',
    headers=headers
)

if response.status_code == 200:
    data = response.json()
    print(f"Encontrados {len(data['data'])} usuários")
else:
    print(f"Erro: {response.status_code}")
JavaScript 
// Usando fetch
async function getUsers() {
  try {
    const response = await fetch('https://api.exemplo.com/v1/users', {
      method: 'GET',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      }
    });
    
    if (!response.ok) {
      throw new Error(`HTTP error! status: ${response.status}`);
    }
    
    const data = await response.json();
    console.log('Usuários:', data.data);
  } catch (error) {
    console.error('Erro:', error);
  }
}

getUsers();
PHP 
<?php
$apiKey = 'YOUR_API_KEY';
$url = 'https://api.exemplo.com/v1/users';

$headers = [
    'Authorization: Bearer ' . $apiKey,
    'Content-Type: application/json'
];

$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $url);
curl_setopt($ch, CURLOPT_HTTPHEADER, $headers);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode === 200) {
    $data = json_decode($response, true);
    echo "Usuários encontrados: " . count($data['data']);
} else {
    echo "Erro: " . $httpCode;
}
?>
